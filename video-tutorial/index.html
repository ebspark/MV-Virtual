How to Embed a Video into Your Website

Follow these simple steps to embed a video hosted on a video-sharing service into your HTML webpage.

Step-by-Step Instructions

Step 1: Find a Video

Navigate to your favorite video-sharing website (e.g., YouTube) and find the video you want to embed.

Step 2: Locate the Embed Option

Below the video, click on the "Share" button, and then select "Embed." This will open a popup with the embed code.

Step 3: Copy the Embed Code

Copy the embed code provided in the popup. It should look like this:

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

Step 4: Paste the Code into Your HTML

Open your HTML file in a text editor, and paste the embed code where you want the video to appear on your webpage. For example:

<body>
    <h1>My Embedded Video</h1>
    <iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>
</body>

Step 5: Save and View Your Page

Save your HTML file and open it in a web browser. You should see the video embedded and ready to play!
